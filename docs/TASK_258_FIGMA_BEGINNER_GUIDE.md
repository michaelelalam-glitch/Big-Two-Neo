### Step 4: Learn the Essential Tools (10 minutes)

**Tool #1: Move Tool (V)**
- **What it does:** Select and move objects
- **How to use:** 
  - Press `V` on keyboard (or click top-left arrow icon)
  - Click any object to select it
  - Drag to move it
  - This is your default toolâ€”you'll use it constantly

**Tool #2: Frame Tool (F)**
- **What it does:** Creates containers (like phone screens)
- **How to use:**
  - Press `F` on keyboard
  - Click and drag on canvas to create a frame
  - OR: Click canvas and select preset (iPhone 14 Pro, etc.)
  - **Think of frames as "screens" in your app**

**Tool #3: Rectangle Tool (R)**
- **What it does:** Creates rectangles (for buttons, cards, etc.)
- **How to use:**
  - Press `R` on keyboard
  - Click and drag on canvas
  - Hold `Shift` while dragging to create a perfect square

**Tool #4: Text Tool (T)**
- **What it does:** Adds text
- **How to use:**
  - Press `T` on keyboard
  - Click anywhere on canvas
  - Start typing
  - Change font/size in right sidebar

**Tool #5: Ellipse Tool (O)**
- **What it does:** Creates circles (for avatars, icons)
- **How to use:**
  - Press `O` on keyboard
  - Click and drag
  - Hold `Shift` for perfect circles

## Common Beginner Mistakes to Avoid

1. **Not using frames:**
   - Always work inside frames (phone screens)
   - Don't place elements directly on canvas

2. **Random colors:**
   - Define your color palette first
   - Stick to your design system colors
   - Don't use random hex codes

3. **Inconsistent spacing:**
   - Use 8pt grid (multiples of 8px)
   - Check spacing between every element

4. **Misaligned elements:**
   - Use Figma's alignment tools
   - Watch for red alignment guides

5. **Too complex on first try:**
   - Start simple, add details later
   - Focus on layout first, polish second

6. **Not naming layers:**
   - Name your frames and groups
   - Makes finding elements easier later

7. **Forgetting mobile constraints:**
   - Design for small screens (393px width)
   - Consider thumb reach zones
   - Test on actual phone if possible

## Troubleshooting

**"I can't see my tools!"**
- Press `V` to activate Move tool
- Tools are on the left sidebar
- If sidebar is hidden: View â†’ Interface â†’ Show Left Sidebar

**"My frame disappeared!"**
- Press `Shift + 1` to zoom to fit all
- Check Layers panel (left sidebar)
- It might be hidden behind another frame

**"I can't select an element!"**
- It might be locked: Right-click â†’ Unlock
- It might be in a group: Double-click to enter group
- Check layer order: It might be behind another element

**"Colors look different on my phone!"**
- Monitors have different color profiles
- Test on actual device before finalizing
- Consider dark/light mode variants

**"Figma is laggy!"**
- Too many effects (shadows, blurs)
- Simplify your design
- Use desktop app instead of browser
- Close other programs
